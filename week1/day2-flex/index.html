Two roles in Flexbox

Every flex property belongs to one of two roles. Knowing which is which stops most of the confusion.

Container properties

Set on the parent. Control how children are arranged.

  • display: flex
  • flex-direction
  • justify-content
  • align-items
  • flex-wrap
  • gap

Item properties

Set on the children. Control how each child behaves.

  • flex-grow
  • flex-shrink
  • flex-basis
  • flex (shorthand)
  • align-self
  • order

7. flex shorthand

flex: grow shrink basis — the three item properties in one. flex: 1 is shorthand for flex: 1 1 0 and is the most common value you'll write in real projects.

ShorthandExpands toMeaning
flex: 1 flex: 1 1 0% Grow, shrink, start from 0 — equal shares
flex: auto flex: 1 1 auto Grow, shrink, start from natural size
flex: none flex: 0 0 auto Don't grow or shrink — rigid size
flex: 0 1 200px flex: 0 1 200px Don't grow, can shrink, starts at 200px

9. order

Changes the visual order of items without touching the HTML. Default is 0. Lower numbers appear first. Useful for responsive reordering — but don't use it for content that screen readers need in a specific order.
