Fullstack Bootcamp — Day 2: CSS Grid

Flex vs Grid — when to use which

Flexbox

1D — controls items along one axis at a time (row or column).

Best for: navbars, button groups, centering a single item, card rows that wrap naturally.

CSS Grid

2D — controls rows and columns simultaneously.

Best for: page layouts, dashboards, image galleries, any design with explicit rows and columns.

Two roles in Grid

Just like Flexbox, every grid property belongs to one of two roles.

Container properties

Set on the parent. Define the grid structure.

  • display: grid
  • grid-template-columns
  • grid-template-rows
  • grid-template-areas
  • grid-auto-rows / columns
  • grid-auto-flow
  • gap / row-gap / column-gap
  • justify-items
  • align-items
  • place-items

Item properties

Set on the children. Control placement and alignment per item.

  • grid-column
  • grid-row
  • grid-area
  • justify-self
  • align-self
  • place-self
